```

```
なやむ: be worry about (verb)
悩めど:

得られる: receptive helper verb conjugation
えられず: without doing verb (ず conjugation)
```

```
ならば: conditional
さかせる: bloom (ichidan verb)
せつな: moment (noun)

はな: flower (noun)
