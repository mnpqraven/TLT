## あさきゆめみし

```
Original game: 东方永夜抄　～ Imperishable Night
Original title: 竹取飛翔　～ Lunatic Princess
Theme: Stage 6 BOSS - Houraisan Kaguya's (蓬莱山 輝夜) theme
```
```
Second to last line in the Iroha poem
Ref: https://ja.wikipedia.org/wiki/%E3%81%84%E3%82%8D%E3%81%AF%E9%A0%86
```

あさきゆめみし　紺青の宵風が流れ
```
あさきゆめみし: superficial dreams (see TL note) (noun)
                紺: dark blue
                青: blue
                紺青 (こんじょう): deep blue
                        宵 (よい): evening/dusk
                        風 (かぜ): wind
                        宵風 (よいかぜ): evening wind (noun)
                            流れ (ながれ): flow/current (noun)

                            ommited だ
```
```
Superficial dreams and the flow of the evening prussian(deep blue) wind
```
さみしやさみし　人の世は邯鄲の刹那よ
```
さみし: lonely (i-adjective)
や: inclusive "and"

                人 (ひと): human/man (noun)
                    世 (よ): life/society (noun)
                        邯鄲 (かんたん): tree cricket (metaphor to a life of vanity/boredom) (noun)
                              刹那 (せつな): moment (noun)
```
```
Lonely o lonely, the life of a man is but a moment of vanity
```

出会い別れ　人の縁は
```
出会う (であう): meet (verb)
      別 (べつ): different
      別れ (わかれ): seperate/farewell (noun)

            人 (ひと): human/man (noun)
                縁 (えにし): fate/destiny/bond (noun)
```
```
The fate of a farewell bidding man
```
寄せては返す波のよう
```
よせる: come/bring smth near (ichidan verb)
        かえす: return (object) (verb)
            なみ: wave (noun)
              のよう: -like metaphorical particle
```
```
is like the tides that come and go
```
ときに穏やかに　暗く激しく
```
ときに: sometimes (noun)
      穏やか (おだやか): calm/quiet (adjective)

                暗い (くらい): dark/gloomy (i-adjective)
                    激しい (はげしい): violent/intense (i-adjective)
                暗く激しく: dark and fierce (i-adjective connector く)
following clause@next line
```
```
In moments of respite, as well as the murky and fervent
```
何もかもを飲み込んでいく
```
何もかも: anything/everything (noun)
          飲み (のみ): drink (noun)
          飲み込む (のみこむ): gulp (verb)
          飲み込んでいく (て form + いく): go gulp (actively does the action)
following clause@next line
```
```
As i gulp down anything i can
```
時は残酷に終わりを告げ
```
時 (とき): time (noun)
    残酷 (ざんこく): cruel (noun)
        に: noun to adverb transformation
          おわる: end (verb)
          終わり: end (noun)
                  つげる: tell/inform (ichidan verb)
                  告げ: tell/inform (imperative verb)
```
```
It was the mark of a cruel end
```
私だけを置き去りにして
```
わたし: me (feminine) (pronoun)
  だけ: only/just
        置く (おく): place/put (verb)
        去る (さる): leave/quit (verb)
        置き去り: leave behind (ten-ten-hooking noun)
                に: target marker particle
                  して: て form of する
```
```
I'm the only one left behind
```
暮れない太陽　明けない夜
```
くれる: darken (ichidan verb)
暮れない: negative ichidan verb
        太い (ふとい): fat/thick (i-adjective)
        陽 (ひ): sunshine
        太陽 (たいよう): the sun (noun)

              あける: open (ichidan verb)
                    (明ける context: dawn/emit light)
              明けな: negative verb
                      よる: night (noun)
```
```
The never-setting sun and the never-falling night
```
誰を愛してもひとり
```
だれ: who
    あいする: love (someone)/care for (verb)
    愛して: connector て
          ても: even if
            ひとり: oneself
```
```
Do i even have anyone to love ?
```

```
ねえ: hey (has either callout or nagging context)
      なにか: something/anything
          ちかう: swear/vow (verb)
          誓って: surely/swear (adverb)
```
```
But wait, I swear that
```

ずっと果たされない約束をして―。
```
ずっと: always/all along
      はたす: achieve/fulfill (verb)
      果たされる: can achieve/fulfill (causative verb)
      果たされない: negative receptive verb
                  やくそく: promise (noun)
                  して: do (て form of する)
```
```
I'll make a promise I can never fulfill
```

「永遠だ」なんて口にしたら
```
えいえん: eternity/forever (noun)
          なんて: such as/such thing (adverb)
                くち: mouth (noun)
                    したら: past form する + たら
                      たら: (expression describing past even that was unexpected/surprising)
```
```
"This is immortality" if you were to utter such thing
```
その瞬間嘘になる
```
その: that
    瞬間 (しゅんかん): moment/instant (noun)
        嘘 (うそ): lie/error (noun)
          に: target particle
            なる: become (verb)
```
```
That'll be a blatant lie
```
だから言って、何度だって
```
だから: therefore/that's why
      言う (いう): say (verb)
      言って: say and (clause connetor て)
              なんど: how many times
                  だって: "but" context (argument/obection demeanor)
                  contradicts 「だから言った」
```
```
That's why, no matter how many times you say it
```
永遠にそばにいると
```
永遠: eternity (noun)
      そば: one's side
          に: location marker particle
          いる: exist/be (ichidan verb)
                と: clause pairing "and", no 2nd clause implying stated 1st clause is absolute/beseech
```
```
Immortality will always be by my side
```

ねえ、ここで誓って
```
ねえ: hey (has either callout or nagging context)
      ここ: here
        　で: location marker particle で
            ちかう: vow/swear (verb)
            誓って: vow/swear and (clause connecting て)
incompleted clause
```
```
Hey, I'll swear to you right here, right now
```
ずっと果たされない約束を今―。

```
ずっと: always/all along
      はたす: achieve/fulfill (verb)
```
