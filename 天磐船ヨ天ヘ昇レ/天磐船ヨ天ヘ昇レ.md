#### VERSE 1

どれくらい闇は続く？
```
どれ: how
どれくらい: dark
          やみ: darkness (noun)
            に: location particle
              つづく: continue (verb)
how far + in darkness + continue ?
```
```
How far will you continue in the darkness ?
```

最後に触れたあなたの指先
```
さいご: lastly/in conclusion
    に: time/location targeting particle
      ふれる: touch/feel (ichidan verb)
      触れた: touch/feel (past tense verb)
            あなた: you (endearing/lower class)(pronoun)
                    ゆび: finger (noun)
                    さき: ahead (noun)
                    指先: fingertip (noun)
lastly + touch + you + of + fingertip (next line)
```

温度さえも
```
あんど: temperature (noun)
    さえ: if only (particle)
        も: even

temperature + even only
```
```
(Even just the heat) of your fingertips' final touch
```

どれくらい落ちたらいい？
```
どれ: how
どれくらい: how dark
          おちる: fall (ichidan verb)
          落ちた: fell (past tense verb)
                  いい: okay/fine (i-adjective)
how much + fell + okay ?
```
```
How deep did you fall down ?
```
